-> Kann man die Datei Application.class von einem Windows-Rechner auf Linux
   kopieren und dort mit dem Programm java zur Ausführung bringen?

Antwort: 

    Java ist plattformunabhängig, da es bytecode aus dem Quelltext generiert, die dann durch eine Laufzeitumgebung abgearbeitet wird.

-> Welche Software muss auf dem Rechner installiert sein?

Antwort:

    Java Development Kit (jdk)
